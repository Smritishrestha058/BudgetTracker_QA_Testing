# Budget Tracker – QA Testing Project

## Project Overview

This repository contains **QA testing documentation** for the Budget Tracker web application.
The goal of this project is to demonstrate **manual testing skills**, including writing test cases, identifying bugs, and writing bug reports

## Modules Tested

* User Registration
* User Login
* Transaction Management
* Dashboard Updates
* Budget Management

## Testing Activities

The following QA activities were performed:

* Writing **test scenarios and detailed test cases**
* Executing test cases and recording **actual results**
* Identifying and documenting **bugs**
